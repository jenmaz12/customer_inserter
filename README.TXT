PROJECT TITLE: Customer Inserter
PURPOSE OF PROJECT: Insert Customers into a Derby database and use DB Viewer to query the database.
VERSION or DATE: 8/16/2020
HOW TO START THIS PROJECT: Compile CustomerInserter.java and DBViewer.java and run main in CustomerInserter and DBViewer.
AUTHORS: Jennifer Mazanec
USER INSTRUCTIONS: Enter customer information in CustomerInserter form and click Add button. Enter a SQL select query in DBViewer text box to query the customer table in the database.
